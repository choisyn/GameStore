# 游戏社区商城系统 - 部署说明文档

## 环境要求

### 开发环境
- **JDK**: 21+
- **Maven**: 3.8+
- **MySQL**: 8.0+
- **Redis**: 6.0+
- **Node.js**: 16+ (如需前端构建)

### 生产环境
- **服务器**: Linux (推荐 Ubuntu 20.04+)
- **内存**: 最小 2GB，推荐 4GB+
- **存储**: 最小 20GB，推荐 50GB+
- **网络**: 公网IP，开放 80/443 端口

## 本地开发环境搭建

### 1. 克隆项目
```bash
git clone <repository-url>
cd game-community-store
```

### 2. 数据库配置
```bash
# 创建数据库
mysql -u root -p
CREATE DATABASE IF NOT EXISTS bishe CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'gamestore'@'localhost' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON bishe.* TO 'gamestore'@'localhost';
FLUSH PRIVILEGES;
```

### 3. Redis配置
```bash
# 启动Redis服务
redis-server

# 或使用Docker
docker run -d --name redis -p 6379:6379 redis:6-alpine
```

### 4. 应用配置
创建 `src/main/resources/application-dev.properties`:
```properties
# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/bishe?useUnicode=true&characterEncoding=utf8mb4&useSSL=false&serverTimezone=Asia/Shanghai
spring.datasource.username=gamestore
spring.datasource.password=password

# Redis配置
spring.redis.host=localhost
spring.redis.port=6379
spring.redis.password=
spring.redis.database=0

# JWT配置
jwt.secret=your-secret-key-here
jwt.expiration=86400000

# 文件上传配置
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=10MB
file.upload.path=/uploads/
```

### 5. 启动应用
```bash
# 使用Maven启动
mvn spring-boot:run -Dspring-boot.run.profiles=dev

# 或编译后启动
mvn clean package
java -jar target/game-community-store-1.0.0.jar --spring.profiles.active=dev
```

## Docker 部署

### 1. 创建 Dockerfile
```dockerfile
FROM openjdk:21-jdk-slim

WORKDIR /app

COPY target/game-community-store-1.0.0.jar app.jar

EXPOSE 8080

ENTRYPOINT ["java", "-jar", "app.jar"]
```

### 2. 创建 docker-compose.yml
```yaml
version: '3.8'

services:
  app:
    build: .
    ports:
      - "8080:8080"
    environment:
      - SPRING_PROFILES_ACTIVE=prod
      - SPRING_DATASOURCE_URL=jdbc:mysql://mysql:3306/bishe?useUnicode=true&characterEncoding=utf8mb4&useSSL=false&serverTimezone=Asia/Shanghai
      - SPRING_DATASOURCE_USERNAME=gamestore
      - SPRING_DATASOURCE_PASSWORD=password
      - SPRING_REDIS_HOST=redis
    depends_on:
      - mysql
      - redis
    volumes:
      - ./uploads:/uploads

  mysql:
    image: mysql:8.0
    environment:
      - MYSQL_ROOT_PASSWORD=rootpassword
      - MYSQL_DATABASE=bishe
      - MYSQL_USER=gamestore
      - MYSQL_PASSWORD=password
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql
      - ./init.sql:/docker-entrypoint-initdb.d/init.sql

  redis:
    image: redis:6-alpine
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./ssl:/etc/nginx/ssl
    depends_on:
      - app

volumes:
  mysql_data:
  redis_data:
```

### 3. 构建和启动
```bash
# 构建应用
mvn clean package

# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f app
```

## 生产环境部署

### 1. 服务器准备
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装必要软件
sudo apt install -y openjdk-21-jdk mysql-server redis-server nginx

# 安装Docker (可选)
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
```

### 2. 数据库配置
```bash
# 配置MySQL
sudo mysql_secure_installation

# 创建数据库和用户
sudo mysql -u root -p
CREATE DATABASE IF NOT EXISTS bishe CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'gamestore'@'localhost' IDENTIFIED BY 'strong_password_here';
GRANT ALL PRIVILEGES ON bishe.* TO 'gamestore'@'localhost';
FLUSH PRIVILEGES;
```

### 3. 应用配置
创建 `application-prod.properties`:
```properties
# 服务器配置
server.port=8080
server.servlet.context-path=/

# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/bishe?useUnicode=true&characterEncoding=utf8mb4&useSSL=false&serverTimezone=Asia/Shanghai
spring.datasource.username=gamestore
spring.datasource.password=${DB_PASSWORD}

# 连接池配置
spring.datasource.hikari.maximum-pool-size=20
spring.datasource.hikari.minimum-idle=5

# Redis配置
spring.redis.host=localhost
spring.redis.port=6379
spring.redis.password=${REDIS_PASSWORD}

# JWT配置
jwt.secret=${JWT_SECRET}
jwt.expiration=86400000

# 日志配置
logging.level.com.gamestore=INFO
logging.file.name=/var/log/gamestore/app.log

# 生产环境配置
spring.jpa.hibernate.ddl-auto=validate
spring.jpa.show-sql=false
```

### 4. 系统服务配置
创建 `/etc/systemd/system/gamestore.service`:
```ini
[Unit]
Description=Game Community Store Application
After=network.target mysql.service redis.service

[Service]
Type=simple
User=gamestore
WorkingDirectory=/opt/gamestore
ExecStart=/usr/bin/java -jar -Xmx2g -Xms1g -Dspring.profiles.active=prod /opt/gamestore/app.jar
Restart=always
RestartSec=10
Environment=DB_PASSWORD=your_db_password
Environment=REDIS_PASSWORD=your_redis_password
Environment=JWT_SECRET=your_jwt_secret

[Install]
WantedBy=multi-user.target
```

### 5. Nginx 配置
创建 `/etc/nginx/sites-available/gamestore`:
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    # 重定向到HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name your-domain.com;
    
    # SSL配置
    ssl_certificate /etc/ssl/certs/your-domain.crt;
    ssl_certificate_key /etc/ssl/private/your-domain.key;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    
    # 安全头
    add_header X-Frame-Options DENY;
    add_header X-Content-Type-Options nosniff;
    add_header X-XSS-Protection "1; mode=block";
    
    # 静态文件
    location /static/ {
        alias /opt/gamestore/static/;
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    # 上传文件
    location /uploads/ {
        alias /opt/gamestore/uploads/;
        expires 1y;
    }
    
    # 应用代理
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # 超时配置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }
}
```

### 6. 启动服务
```bash
# 创建用户和目录
sudo useradd -r -s /bin/false gamestore
sudo mkdir -p /opt/gamestore /var/log/gamestore
sudo chown gamestore:gamestore /opt/gamestore /var/log/gamestore

# 部署应用
sudo cp target/game-community-store-1.0.0.jar /opt/gamestore/app.jar
sudo chown gamestore:gamestore /opt/gamestore/app.jar

# 启动服务
sudo systemctl daemon-reload
sudo systemctl enable gamestore
sudo systemctl start gamestore

# 配置Nginx
sudo ln -s /etc/nginx/sites-available/gamestore /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# 检查服务状态
sudo systemctl status gamestore
sudo systemctl status nginx
```

## 监控和维护

### 1. 日志管理
```bash
# 查看应用日志
sudo tail -f /var/log/gamestore/app.log

# 日志轮转配置 /etc/logrotate.d/gamestore
/var/log/gamestore/*.log {
    daily
    missingok
    rotate 30
    compress
    delaycompress
    notifempty
    copytruncate
}
```

### 2. 性能监控
```bash
# 系统资源监控
htop
iostat -x 1
free -h

# 应用监控
curl http://localhost:8080/actuator/health
curl http://localhost:8080/actuator/metrics
```

### 3. 数据库维护
```bash
# 数据库备份
mysqldump -u gamestore -p bishe > backup_$(date +%Y%m%d).sql

# 定期备份脚本
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
mysqldump -u gamestore -p$DB_PASSWORD bishe | gzip > /backup/gamestore_$DATE.sql.gz
find /backup -name "gamestore_*.sql.gz" -mtime +7 -delete
```

### 4. 更新部署
```bash
# 停止服务
sudo systemctl stop gamestore

# 备份当前版本
sudo cp /opt/gamestore/app.jar /opt/gamestore/app.jar.backup

# 部署新版本
sudo cp target/game-community-store-1.0.1.jar /opt/gamestore/app.jar
sudo chown gamestore:gamestore /opt/gamestore/app.jar

# 启动服务
sudo systemctl start gamestore

# 检查状态
sudo systemctl status gamestore
curl http://localhost:8080/actuator/health
```

## 故障排除

### 常见问题
1. **应用启动失败**: 检查日志、数据库连接、端口占用
2. **数据库连接失败**: 检查用户权限、网络连接
3. **Redis连接失败**: 检查Redis服务状态、配置
4. **文件上传失败**: 检查目录权限、磁盘空间

### 应急处理
```bash
# 快速重启
sudo systemctl restart gamestore

# 回滚版本
sudo cp /opt/gamestore/app.jar.backup /opt/gamestore/app.jar
sudo systemctl restart gamestore

# 清理缓存
redis-cli FLUSHALL
```
