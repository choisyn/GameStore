## API接口总览

### 游戏相关
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/games` | 获取游戏列表 |
| GET | `/api/games/{id}` | 获取游戏详情 |
| GET | `/api/games/featured` | 获取精选游戏 |
| GET | `/api/games/search` | 搜索游戏 |

### 用户认证
| 方法 | 路径 | 描述 |
|------|------|------|
| POST | `/api/auth/register` | 用户注册 |
| POST | `/api/auth/login` | 用户登录 |
| POST | `/api/auth/logout` | 用户登出 |
| GET | `/api/auth/current` | 获取当前用户 |

### 社区相关
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/community/sections` | 获取板块列表 |
| GET | `/api/community/sections/{id}/posts` | 获取板块帖子 |
| POST | `/api/community/posts` | 发布帖子 |
| POST | `/api/community/posts/{id}/comments` | 发布评论 |

### 后台管理 - 游戏
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/admin/games` | 获取所有游戏 |
| POST | `/api/admin/games` | 创建游戏 |
| PUT | `/api/admin/games/{id}` | 更新游戏 |
| DELETE | `/api/admin/games/{id}` | 删除游戏 |

### 后台管理 - 用户
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/admin/users` | 获取所有用户 |
| PUT | `/api/admin/users/{id}` | 更新用户 |
| PUT | `/api/admin/users/{id}/role` | 修改角色 |
| PUT | `/api/admin/users/{id}/status` | 修改状态 |
| PUT | `/api/admin/users/{id}/reset-password` | 重置密码 |
| GET | `/api/admin/users/stats` | 用户统计 |

### 后台管理 - 社区
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/admin/community/posts` | 获取所有帖子 |
| PUT | `/api/admin/community/posts/{id}/pin` | 置顶帖子 |
| PUT | `/api/admin/community/posts/{id}/essence` | 设为精华 |
| DELETE | `/api/admin/community/posts/{id}` | 删除帖子 |
| DELETE | `/api/admin/community/comments/{id}` | 删除评论 |

## 已知问题

### 安全问题
⚠️ **密码未加密** - 当前密码以明文存储，需要使用BCrypt加密  
⚠️ **缺少CSRF保护** - 表单提交未添加CSRF Token  
⚠️ **缺少XSS防护** - 用户输入未进行HTML转义

### 功能问题
⚠️ **图片上传** - 当前只能使用URL，无法直接上传图片  
⚠️ **分页优化** - 后台管理分页功能不完善  
⚠️ **错误处理** - 部分异常处理不够友好
