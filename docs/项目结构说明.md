# 游戏社区商城系统 - 项目结构说明

## 项目概述
**项目名称**: GameCommunityStore (游戏社区商城系统)  
**版本**: 1.0.0  
**技术栈**: Spring Boot 3.2.0 + Spring Data JPA + MySQL + Thymeleaf + Bootstrap 5  
**Java版本**: 23  
**UI风格**: Steam商城风格  
**数据库**: MySQL 8.0 (bishe数据库)

## 数据库架构

### 核心表结构

#### 1. users - 用户表
```sql
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    avatar VARCHAR(200),
    role ENUM('USER', 'ADMIN', 'MODERATOR'),
    status ENUM('ACTIVE', 'INACTIVE', 'BANNED'),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    last_login_at TIMESTAMP
);
```

#### 2. games - 游戏表 ✨已更新
```sql
CREATE TABLE games (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category_id BIGINT,
    developer VARCHAR(100),
    publisher VARCHAR(100),
    release_date DATE,
    price DECIMAL(10,2) DEFAULT 0,
    discount_price DECIMAL(10,2),
    image_url VARCHAR(500),
    gallery TEXT,
    system_requirements TEXT,           -- 文本格式存储
    tags VARCHAR(200),                  -- 逗号分隔的标签
    rating DECIMAL(3,2) DEFAULT 0,
    rating_count INT DEFAULT 0,
    download_count INT DEFAULT 0,
    is_featured BOOLEAN DEFAULT FALSE,
    status ENUM('ACTIVE', 'INACTIVE', 'COMING_SOON'),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 3. categories - 分类表
```sql
CREATE TABLE categories (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    description VARCHAR(200),
    parent_id BIGINT,
    sort_order INT DEFAULT 0,
    icon_url VARCHAR(200),
    status ENUM('ACTIVE', 'INACTIVE'),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 实体类映射

### Game实体（游戏）
```java
@Entity
@Table(name = "games")
public class Game {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    @Column(nullable = false, length = 100)
    private String name;
    
    @Column(columnDefinition = "TEXT")
    private String description;
    
    @Column(name = "category_id")
    private Long categoryId;
    
    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "category_id", insertable = false, updatable = false)
    private Category category;
    
    private String developer;
    private String publisher;
    
    @Column(name = "release_date")
    private LocalDate releaseDate;
    
    @Column(precision = 10, scale = 2)
    private BigDecimal price;
    
    @Column(name = "discount_price", precision = 10, scale = 2)
    private BigDecimal discountPrice;
    
    @Column(name = "image_url", length = 500)
    private String imageUrl;
    
    @Column(columnDefinition = "TEXT")
    private String systemRequirements;  // 文本格式
    
    @Column(length = 200)
    private String tags;  // 逗号分隔的字符串
    
    // ... 其他字段和方法
}
```

## 业务流程

### 用户浏览游戏流程
```
1. 访问首页 (/)
   ↓
2. 查看游戏列表（轮播图 + 网格）
   ↓
3. 点击游戏卡片
   ↓
4. 跳转到详情页 (/game/{id})
   ↓
5. 查看完整游戏信息
   ↓
6. 加入购物车/添加愿望单
```

### 数据流程
```
前端请求
   ↓
Controller（控制器层）
   ↓
Service（业务逻辑层）
   ↓
Repository（数据访问层）
   ↓
MySQL数据库
   ↓
返回Entity对象
   ↓
Thymeleaf渲染/JSON响应
   ↓
返回给前端
```
